## 역할 계층

```
OWNER  >  MANAGER  >  MEMBER
```

상위 역할은 하위 역할의 모든 권한을 포함합니다.

## 권한 표

| 작업 | MEMBER | MANAGER | OWNER |
|---|:---:|:---:|:---:|
| 게시글 읽기 | ✅ | ✅ | ✅ |
| 게시글 작성 | ✅ | ✅ | ✅ |
| 댓글 작성 | ✅ | ✅ | ✅ |
| 본인 글 수정/삭제 | ✅ | ✅ | ✅ |
| 타인 글 삭제 | — | ✅ | ✅ |
| 멤버 초대 | — | ✅ | ✅ |
| 비밀번호 초기화 | — | ✅ | ✅ |
| 스페이스/게시판 생성 | — | — | ✅ |
| 스페이스/게시판 삭제 | — | — | ✅ |
| 역할 변경 | — | — | ✅ |
| 설정 변경 | — | — | ✅ |

## 주의사항

- 역할은 **스페이스**와 **게시판** 단위로 개별 설정됩니다
- 스페이스 OWNER가 자동으로 하위 게시판의 OWNER가 되지는 않습니다
- OWNER 역할 변경은 OWNER만 가능합니다
