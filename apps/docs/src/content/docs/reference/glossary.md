---
title: 용어 설명
description: blommunity에서 사용하는 주요 용어를 정리합니다.
---

| 용어 | 설명 |
|---|---|
| **테넌트** | blommunity 위에서 운영되는 독립된 커뮤니티 단위 |
| **스페이스** | 테넌트 안에서 주제나 그룹을 나누는 공간 |
| **게시판** | 스페이스 안에서 글이 올라오는 공간 |
| **멤버** | 커뮤니티에 가입한 사용자 |
| **운영자** | blommunity 플랫폼 전체를 관리하는 내부 관리자 |
| **PUBLIC** | 모든 멤버에게 공개된 상태 |
| **PRIVATE** | 초대받은 멤버만 접근 가능한 상태 |
| **OWNER** | 최고 권한을 가진 역할 |
| **MANAGER** | 중간 관리자 역할 |
| **MEMBER** | 일반 멤버 역할 |
| **PENDING** | 초대가 아직 수락되지 않은 대기 상태 |
| **ACTIVE** | 초대가 수락되어 활성화된 멤버십 상태 |
| **테넌트 콘솔** | 테넌트 운영자가 커뮤니티를 관리하는 CMS |
| **커뮤니티 사이트** | 일반 멤버가 이용하는 커뮤니티 화면 |
