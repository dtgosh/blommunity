## 게시글 작성

1. 게시판 목록에서 원하는 게시판을 선택합니다.
2. **"글쓰기"** 버튼을 클릭합니다.
3. **제목**과 **내용**을 입력합니다.
4. **등록** 버튼을 클릭합니다.

:::note
게시글은 MEMBER 이상의 역할이 있어야 작성할 수 있습니다.
:::

## 댓글 작성

게시글 하단의 댓글 입력창에 내용을 입력하고 **전송** 버튼을 클릭합니다.

## 대댓글

댓글 우측의 **답글** 버튼을 클릭하면 해당 댓글에 대한 대댓글을 작성할 수 있습니다.

## 게시글/댓글 수정 및 삭제

본인이 작성한 게시글과 댓글은 수정하거나 삭제할 수 있습니다. 게시글/댓글 우측의 더보기 메뉴(···)를 클릭하세요.
