운영자로부터 비공개 스페이스 또는 게시판에 초대를 받으면 수락해야 접근할 수 있습니다.

## 초대 수락

<Steps>

1. 상단 메뉴에서 **초대함** 아이콘을 클릭합니다.

2. 대기 중인 초대 목록이 표시됩니다.

3. 원하는 초대의 **수락** 버튼을 클릭합니다.

4. 수락 후 해당 스페이스/게시판에 즉시 접근할 수 있습니다.

</Steps>

## 초대 거절

초대를 거절할 수도 있습니다. 거절한 초대는 다시 수락할 수 없으며, 운영자가 재초대해야 합니다.

:::tip
초대를 수락하지 않으면 해당 PRIVATE 스페이스는 목록에 표시되지 않습니다.
:::
