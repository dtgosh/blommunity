## 테넌트 (Tenant)

하나의 독립된 커뮤니티입니다. 테넌트마다 별도의 멤버, 스페이스, 게시판이 있으며 다른 테넌트와 데이터가 분리됩니다.

- 테넌트 ID로 구분됩니다
- 운영자가 승인을 통해 개설합니다

## 스페이스 (Space)

테넌트 안에서 주제나 그룹을 나누는 공간입니다.

| 공개 범위 | 설명 |
|---|---|
| **PUBLIC** | 모든 멤버가 볼 수 있습니다 |
| **PRIVATE** | 초대받은 멤버만 접근할 수 있습니다 |

## 게시판 (Board)

스페이스 안에 있는 실제 게시판입니다. 각 게시판도 PUBLIC / PRIVATE 설정이 가능합니다.

## 역할 (Role)

모든 멤버는 스페이스와 게시판별로 역할을 가집니다.

| 역할 | 권한 |
|---|---|
| **OWNER** | 스페이스/게시판 설정 변경, 멤버 역할 변경, 삭제 |
| **MANAGER** | 멤버 초대, 게시글 관리 |
| **MEMBER** | 게시글 읽기, 작성, 댓글 |

:::note
역할은 계층 구조입니다. OWNER는 MANAGER와 MEMBER의 모든 권한을 가집니다.
:::

## 멤버십 상태

초대를 받으면 `PENDING` 상태로 대기합니다. 초대를 수락하면 `ACTIVE`가 됩니다.
