title: blommunity란?

---

blommunity는 **어떤 조직이든 자신만의 커뮤니티를 운영**할 수 있게 해주는 게시판 기반 플랫폼입니다.

## 누가 사용하나요?

blommunity에는 두 종류의 사용자가 있습니다.

### 테넌트 운영자

독서 모임, 스터디 그룹, 기업 등 커뮤니티를 운영하는 사람입니다. **테넌트 콘솔**에서 다음을 관리합니다.

- 스페이스(공간)와 게시판 생성
- 멤버 초대 및 역할 부여
- 공개/비공개 설정

### 커뮤니티 멤버

운영자의 커뮤니티에 참여하는 사람입니다. **커뮤니티 사이트**에서 다음을 할 수 있습니다.

- 게시글 읽기, 작성, 댓글
- 초대받은 비공개 스페이스 참여
- 프로필 관리

## 기본 구조

```
테넌트 (커뮤니티 전체)
└── 스페이스 (주제별 공간)
    └── 게시판 (카테고리별 게시판)
        └── 게시글 + 댓글
```

예를 들어 **북클럽** 테넌트라면:

- 스페이스: "월간 독서", "자유 토론"
- 게시판: "이달의 책", "독서 기록", "질문과 답변"
