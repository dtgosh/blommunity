## 멤버 목록 확인

좌측 메뉴에서 **멤버**를 클릭하면 현재 커뮤니티의 전체 멤버 목록을 볼 수 있습니다.

## 역할 변경

멤버의 역할은 **OWNER만** 변경할 수 있습니다.

1. 멤버 목록에서 변경할 멤버를 클릭합니다.
2. **역할 변경** 버튼을 클릭합니다.
3. 새 역할을 선택합니다.

| 역할 | 설명 |
|---|---|
| **OWNER** | 모든 권한 — 역할 변경, 삭제, 설정 |
| **MANAGER** | 멤버 초대, 게시글 관리 |
| **MEMBER** | 읽기, 게시글/댓글 작성 |

:::caution
OWNER 역할은 신중하게 부여하세요. OWNER는 다른 OWNER의 역할도 변경할 수 있습니다.
:::

## 비밀번호 초기화

멤버가 비밀번호를 잊었을 경우, MANAGER 이상의 권한이 있는 운영자가 비밀번호를 초기화할 수 있습니다.

1. 멤버 상세 페이지에서 **비밀번호 초기화**를 클릭합니다.
2. 임시 비밀번호를 설정합니다.
3. 멤버에게 임시 비밀번호를 전달합니다.
