스페이스는 커뮤니티를 주제나 그룹 단위로 나누는 공간입니다.

## 스페이스 만들기

<Steps>

1. 좌측 메뉴에서 **스페이스**를 클릭합니다.

2. 우측 상단의 **"+ 새 스페이스"** 버튼을 클릭합니다.

3. 다음 정보를 입력합니다.
   - **이름** — 스페이스 이름 (예: "월간 독서", "자유 토론")
   - **설명** (선택) — 스페이스 소개
   - **공개 범위** — PUBLIC 또는 PRIVATE

4. **만들기** 버튼을 클릭합니다.

</Steps>

## 공개 범위 설정

| 설정 | 설명 |
|---|---|
| **PUBLIC** | 모든 멤버가 스페이스를 볼 수 있습니다 |
| **PRIVATE** | 초대받은 멤버만 접근할 수 있습니다 |

:::tip
비공개 스페이스는 초대받은 멤버에게만 보입니다. 운영진 전용 공간이나 소규모 그룹에 활용하세요.
:::

## 멤버 초대하기

PRIVATE 스페이스에는 멤버를 직접 초대해야 합니다.

1. 스페이스 상세 페이지에서 **멤버** 탭을 클릭합니다.
2. **"멤버 초대"** 버튼을 클릭합니다.
3. 초대할 멤버 아이디를 입력합니다.
4. 초대된 멤버가 수락하면 스페이스에 참여됩니다.

## 스페이스 수정 및 삭제

스페이스 상세 페이지 우측 상단의 메뉴에서 수정하거나 삭제할 수 있습니다.

:::caution
스페이스를 삭제하면 하위 게시판과 모든 게시글이 함께 삭제됩니다. 되돌릴 수 없으니 신중하게 결정해주세요.
:::
