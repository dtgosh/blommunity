게시판은 스페이스 안에서 실제로 글이 올라오는 공간입니다.

## 게시판 만들기

<Steps>

1. 좌측 메뉴에서 **게시판**을 클릭합니다.

2. **"+ 새 게시판"** 버튼을 클릭합니다.

3. 다음 정보를 입력합니다.
   - **이름** — 게시판 이름 (예: "이달의 책", "자유게시판")
   - **스페이스** — 어느 스페이스에 속할지 선택
   - **설명** (선택)
   - **공개 범위** — PUBLIC 또는 PRIVATE

4. **만들기**를 클릭합니다.

</Steps>

## 게시판 공개 범위

게시판도 스페이스와 독립적으로 공개 범위를 설정할 수 있습니다.

:::note
PRIVATE 스페이스 안의 PUBLIC 게시판은 **스페이스 멤버에게만** 공개됩니다. 스페이스 접근 권한이 먼저 확인됩니다.
:::

## 멤버 관리

게시판별로 멤버를 개별 초대하고 역할을 부여할 수 있습니다. 이를 활용해 특정 게시판에만 쓰기 권한을 제한할 수 있습니다.
